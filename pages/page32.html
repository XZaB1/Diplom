SQL Server Reporting Services

Программная серверная система создания отчётов, разработанная корпорацией Microsoft. Она может быть использована для подготовки множества интерактивных и печатных отчётов. Система администрируется через веб-интерфейс.

Службы SQL Server Reporting Services (SSRS)предоставляют набор локальных средств и служб для создания и развертывания отчетов с разбиением на страницы, а также управления ими. На скриншоте – выгрузка ошибок конвертации в Адакту.
